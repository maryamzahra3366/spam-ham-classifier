# spam-ham-classifier
A simple machine learning project to classify text messages as spam or ham using Python and scikit-learn.
Dataset

CSV file with two columns: label (spam/ham) and text (message content).

## Usage

1. Open the Jupyter Notebook:

```bash
jupyter notebook SpamClassifier.ipynb
Follow the cells in order:

Load dataset

Split data

Vectorize text

Train models and tune hyperparameters

Evaluate models

Find the best model

The notebook prints the best model at the end:

Best model: RandomForestClassifier
